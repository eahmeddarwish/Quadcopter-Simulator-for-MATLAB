# AprilNav

A configurable indoor quadcopter navigation & AprilTag simulation
toolbox for MATLAB/Simulink. This Space is a **static showcase** —
the toolbox itself runs locally in MATLAB/Simulink (a hosted Space
cannot run MATLAB), so this page exists to explain the project, show
what it does, and link back to the source.

**Source code:** see the linked GitHub repository below (Ahmed Darwish
will add the link once pushed).
